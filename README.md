# mind-body-soul
A website for information on mindfulness, yoga and meditation  
 # About

This website has been created to provide a rundown of the basics and background of mindfulness, yoga and meditation and also to inform the user in the wide range of benefits that these practices can provide.

## User Experience (UX)
### User Stories
- A user that is stressed and wants to find out if  mindfulness, yoga or meditation could help
1. I want the layout of the website to be easy to navigate and not cluttered.
2. I want to have enough information so that I am informed but not so much that I feel overwhelmed 
3. I want to have the option to contact the comapany for further information
4. I want to be able to open external links to learn further infomation on each activity.

## Testing

### Functionality Testing

#### Navigation Bar

-When the main heading of “Mind, Body & Soul” is clicked, it will direct the user back to the home page, which has been tested on all pages and screen sizes.

-All five navigational links all respond correctly and direct the user to the desired page. This again has been tested on all pages and on all screen sizes.

-The active page will display an underline, to inform the user of their current position on the site.

-The inactive page links will display an underline when hovered over.

-On screen sizes 950px wide plus, the navigational bar will appear to the right of the main heading.

-Screen sizes from 950px to 601px, the navigational bar will appear below the main heading with a reduction in font size.

-Screen sizes 600px and below, the main heading will reduce in size and the navigational links will sit below, one stacked on top of the other.

#### Hero Image

-The hero images on all pages have a zoom function when first loaded, this is used on all pages and screen sizes.

-Cover text is present on all hero images which sits to the right of the screen for screens 950px wide plus.

-Cover text for screen sizes 950px wide and below will sit to the left of the screen and eventually sit in the center as the screen size decreases.

#### Main sections

-All lists, paragraphs and images will sit level with one another in their respective sections for screen sizes 1200px wide plus, below 1200px they will begin to stack themselves and widen to allow for better readability.

-Video links displayed in iframes will respond to various screen sizes in order to avoid an elongated or stretched viewing for the user.  

#### Footer

-Social icons, Contact link and Google maps link font size will reduce in size for screen sizes 800px and below to fit viewing area.

-All links provided in footer section have been tested and will open in a new tab.
